oracle 新建数据库和用户

2022/8/17 傻瓜笔记 sql oracle

背景

在开发的时候常常需要对不同的项目做分库的处理，在这里记录下oracle新建数据库和用户的sql语句

得到数据库常用存放位置

个人喜欢数据库文件放在同一个文件夹下，如果没特殊要求，可跳过这一步

select name from v$datafile

新建数据库

CREATE TABLESPACE 数据库名称 LOGGING DATAFILE '数据库文件存放位置.DBF'
SIZE 100M AUTOEXTEND ON NEXT 32M MAXSIZE 200M EXTENT MANAGEMENT LOCAL;

创建用户、密码并分配数据库

CREATE USER 用户名 IDENTIFIED BY 密码 DEFAULT TABLESPACE 数据库名称;

分配用户权限

grant connect,resource to 用户名称;

用户权限说明

1.dba: 拥有全部特权，是系统最高权限，能够新建数据库。
2.resource:能够增删改查、新建表实体。
3.connect:能够登录到数据库。

一般情况下：

​ 普通用户：resource、connect

​ 有管理数据库需求的用户：dba、resource、connect

！！！

其他未说明的权限请自行百度，并自行分配用户权限

注意事项

！！！

在执行以上sql语句时用户需要有dba权限
