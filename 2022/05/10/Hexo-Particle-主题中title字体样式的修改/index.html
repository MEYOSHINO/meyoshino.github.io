Hexo Particle 主题中title字体样式的修改

2022/5/10 解决方案 hexo particle ui 字体样式修改

在使用Hexo框架+Particle主题搭建个人博客网站时，总觉得某些区域的字体怪怪的，于是决定给字换换样式，顺便记录下修改过程，给之后同样想修改字体样式的同学一个参考

下载字体

这里使用Google-fonts下载字体，下载得到的是一个字体的压缩包，解压得到字体文件，所有的字体文件后缀名都是.tff

字体引入

将需要的字体文件拷贝至.\themes\particle\source\css\fonts文件夹下，并在.\themes\particle\source\css\defalut.styl文件中添加以下内容，fontName为自定义名字

@font-face
  font-family 'fontName'
  src url('../css/fonts/fontFileName.ttf')

字体全局声明

将字体注册成全局变量，方便在css文件中使用，在.\themes\particle\source\css\full-color.styl文件末尾添加以下内容，需要注意的是这里的fontName应该与default.styl中的定义的一致，XXXXXX-font-family可自定义命名，之后在使用时就只需要这个名字

XXXXXX-font-family = fontName

字体的使用

运行Hexo项目，在页面中通过F12，找到想要修改样式的地方对应项目中的css样式文件的位置，如这里我想修改tiltle的字体样式，可以在浏览器调试窗口找到如下这些关键信息

通过ide搜索发现其样式信息在.\themes\particle\source\css\layout.styl文件下，修改样式信息，增加font-family XXXXXX-font-family,这里的XXXXXX-font-family即上文填写的全局变量名

效果展示

在项目根目录下运行以下指令，我们就可以发现字体样式被改变了

hexo clean && hexo g && hexo s

总结

以上的方法只适应某一块小区域上的修改，如果要大范围的修改还是有点麻烦的，幸好原来的字体也挺好看的。在这里推荐下所使用的主题Particle，页面十分简洁，还有可爱的喵咪
